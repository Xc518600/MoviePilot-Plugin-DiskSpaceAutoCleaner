# 硬盘空间自动清理

MoviePilot V2 插件：监控指定硬盘/媒体库剩余空间，在空间不足时生成老旧电影或电视剧清理建议。

## v1.0 定位

**只报告，不删除。**

本版本用于安全评估：

- 检查指定监控路径剩余空间
- 空间低于阈值时扫描媒体目录
- 根据文件/目录大小、修改时间生成候选清理列表
- 支持保护目录、保护关键词、最近新增保护
- 保存历史记录
- 可发送通知
- 不删除任何文件，不修改媒体库，不影响 NAS 环境

## 安全原则

- 默认安全报告模式
- 不执行删除
- 不移动文件
- 不清理下载器任务
- 仅生成建议清单
